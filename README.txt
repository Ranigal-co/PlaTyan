PlaTyan - музыкальный аудиоплеер
функции:
- Добавление и удаление музыки через кнопку открытия файлов:
  - При нажатии кнопки (добавить песни) откроется диалоговое окно выбора файлов .mp3 и .wav
  - При нажатии кнопки (удалить песню) удалится выделенная песня из списка
    (нажать курсором на песню в плейлисте и затем нажать кнопку удаления песни)
- Создание и удаление плейлистов:
  - При нажатии кнопки (создать плейлист) добавится пустой плейлист
  - При нажатии кнопки (удалить плейлист) удалится выбранный плейлист
- Прослушивание музыки (кнопка старт/пауза)
- Следующий/предыдущий трэк (кнопки следующий/предыдущий)
- 3 варианта прослушивания:
  - Повтор песни
  - Послушать плейлист 1 раз
  - Повторять плейлист
- Перемотка песни в любую позицию (слайдер песни)
- Изменение громкости (слайдер громкости)

/////////////////////////////////////////////////////////////////////////////////////////////////

Реализация:
3 класса аудиоплеера:
- Audio.py # Описание всех функций приложения
- database.py # Описание базы данных и работы с ней
- struct_player.py # Расположение виджетов на окне

3 папки ресурсов:
- gif # видео анимаций
- icon # иконки
- image # картинки фреймов анимаций

database.db - база данных, содержащая песни и плейлисты в двух таблицах [songs, playlists]

Технологии:
- PySide6 - библиотека для графических приложений
- sqlite3 - библиотека для работы с базами данных
- eyed3 - библиотека импортирующая свойства песен
- Обработка исключений
- Работа с файлами
- Диалоги окон
- Работа с изображениями, видео и аудио
- SQL запросы
- Обработка курсора мыши
- Работа со стилями
